Workflow

Four moves to zero plaintext.

Every secret follows the same lifecycle. Detect, encrypt, mask, resolve. No exceptions.

1

Scan

222+ built-in patterns detect AWS keys, GitHub tokens, private keys, JWTs, database URLs. Runs fully offline — no API calls.

vk scan .env config/
2

Filter

Replace each detected secret with an AES-256-GCM encrypted VK: reference. The original plaintext is destroyed immediately.

vk filter .env
3

Wrap

Execute commands with automatic output masking. If a secret appears in stdout or stderr, it's replaced before reaching the terminal.

vk wrap ./deploy.sh
4

Exec

Resolve VK: tokens at launch time. Decrypted values are injected into process memory — never written to disk or passed as env vars.

vk exec ./my-app

Architecture

Hub-and-spoke. Every node isolated.

One KeyCenter controls policy. LocalVault nodes hold only ciphertext. Compromise a node — the rest stay sealed.

Encryption

Three-layer key hierarchy.

Password → KEK → DEK → ciphertext. Each layer independently rotatable.

KEK→ DEK→ VK:REF
AI Safety

AI sees references. Never real keys.

Claude, Codex, or any agent works with VK: tokens. Everything functions — zero exposure.

Rotation

Zero-downtime key rotation.

KeyCenter schedules. Each node re-encrypts independently with its own DEK. No restarts, no coordination, no downtime. Rotation is a background operation.
