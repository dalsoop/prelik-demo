Components

달 안의 구성 요소

.dalfactory 템플릿이 선언하는 4가지 핵심 구성 요소. 각각 독립적으로 설치·버전 관리·내보내기가 가능합니다.

CLI
CLI 도구
달 안에 설치되는 명령줄 도구 패키지들입니다. cloud hub의 레지스트리에서 DAL:CLI ID로 관리되며, 컨테이너 프로비저닝 시 자동으로 설치됩니다.
# .dalfactory/templates/default.cue cli: [ "DAL:CLI:b1c2d3e4", // dalcli-code-runner "DAL:CLI:f4a91c0e", // dalcli-test-gen ]
SKILL
에이전트 스킬
AI 에이전트가 사용하는 스킬 파일들입니다. .dalfactory/skills/ 에 위치하며, dalcenter export 실행 시 ~/.claude/skills/ 또는 ~/.codex/skills/ 로 심링크됩니다.
# exports 선언 exports: { claude: { skills: [ "skills/code-review/skill.md", "skills/pr-check/skill.md", ] } }
HOOK
이벤트 훅
pre-commit, post-push 등 git 이벤트에 연결되는 훅 스크립트입니다. export 시 ~/.claude/hooks/ 에 심링크되고 settings.json 에 자동 등록됩니다.
# exports.claude.hooks hooks: [ "hooks/pre-commit.sh", "hooks/post-push.sh", ] → ~/.claude/hooks/pre-commit.sh → settings.json 자동 등록
SECRET
SecretVault
AES-256-GCM으로 암호화되어 SQLite에 저장되는 시크릿입니다. .env 파일 없이 컨테이너 기동 시 환경변수로 자동 복호화 주입됩니다.
# 시크릿 등록 $ dalcenter secret set GITHUB_TOKEN ✓ encrypted (AES-256-GCM) → vault # 컨테이너 기동 시 자동 주입 → GITHUB_TOKEN=sk-... (env) → .env 파일 불필요
Export Pipeline

선언 → 내보내기 → AI 런타임

.dalfactory 에 선언된 스킬과 훅은 dalcenter export 한 번으로 각 AI 런타임 홈 디렉토리로 심링크됩니다.

dalcenter export <dal-id> — 내보내기 흐름

Workflow

달 생애주기

step 01
선언
.dalfactory/templates/ 에 CUE 파일로 CLI · 스킬 · 훅 · 시크릿 · 컨테이너를 선언합니다. validate 로 즉시 검증.
$ dalcenter validate
step 02
등록 + 내보내기
join 으로 달 ID 발급 후, export 로 스킬/훅을 AI 런타임(.claude/, .codex/) 에 심링크합니다.
$ dalcenter join && dalcenter export
step 03
프로비저닝
LXC 컨테이너 자동 생성. CLI 도구 설치, PLAYER 기동, SecretVault 시크릿 자동 주입.
$ dalcenter provision <id>
step 04
감시 + 복구
watch 모드 60s 자동 점검. drift 감지 시 reconcile 자동 실행. 에이전트는 꺼지지 않습니다.
$ dalcenter watch
