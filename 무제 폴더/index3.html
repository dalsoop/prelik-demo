CLI Reference
dalcenter 명령어
모든 작업은 dalcenter CLI로 처리한다. 레포 등록부터 LXC 프로비저닝, drift 감지까지.
command description example type
join
.dalfactory manifest를 읽어 인스턴스를 등록한다. 로컬 경로 또는 클라우드 패키지명.
 join .
join agent-coach		 core
validate
dal.spec.cue 기반 manifest 유효성 검증.
 validate . core
export
선언된 Claude 스킬·훅을 .claude/로 export한다.
 export . ops
unexport
export된 스킬·훅을 제거한다.
 unexport . ops
reconcile
전체 인스턴스의 export 상태를 점검하고 drift를 수복한다.
 reconcile ops
watch
interval마다 reconcile을 반복 실행하는 감시 데몬. 싱글톤 PID로 보장.
 watch --interval 60 ops
provision
Proxmox LXC 컨테이너 생성, 패키지·에이전트 설치. --dry-run 지원.
 provision dal --vmid 219 lxc
destroy
프로비저닝된 LXC를 중지·제거한다.
 destroy dal lxc
start / stop
/ restart
로컬 프로세스 관리. build.entry가 있으면 --command 생략 가능.
 start dal ops
secret set/get/list
AES-256-GCM SecretVault에 API 키 등 민감 정보를 저장·조회·나열한다.
 secret set ANTHROPIC_KEY vault
catalog search
dalforge 클라우드 허브에서 dalcli 패키지를 검색한다.
 catalog search agent core
list / status
등록된 인스턴스 목록 및 상세 상태(health, pid, exports 수)를 조회한다.
 list
status my-agent		 core

Quick Start
5분 안에
시작하기
01
빌드 & 설치
Go 1.25+ 환경에서 dalcenter를 빌드한다.
SHELL
# 레포 클론 후 빌드
$ git clone https://github.com/dalsoop/dalforge
$ cd dalforge && go build -o ~/go/bin/dalcenter ./cmd/dalcenter/
✓ ~/go/bin/dalcenter
02
.dalfactory 선언
레포 루트에 .dalfactory/dal.cue를 만든다.
CUE
name: "my-agent"
template: "claude-dev"
container: { base: "ubuntu-24.04" }
agents: { claude: { enable: true } }
03
등록
dalcenter가 manifest를 읽고 인스턴스에 ID를 부여한다.
SHELL
$ dalcenter validate . && dalcenter join .
instance created: DAL:CONTAINER:a1b2c3d4
template=claude-dev skills=3
04
LXC 프로비저닝
Proxmox에 컨테이너를 생성하고 에이전트를 설치한다.
SHELL
$ dalcenter provision my-agent --vmid 219 --storage local-lvm
provisioned (vmid=219)
05
Watch 시작
60초마다 상태를 점검하고 drift를 자동 수복한다.
SHELL
$ dalcenter watch --interval 60
watch started (interval=60s, pid=48291)
DAL:CONTAINER:a1b2c3d4: ok
.dalfactory/dal.cue
// 레포 선언 SSOT
name: "my-agent"
version: "1.0.0"
template: "claude-dev"

container: {
  base: "ubuntu-24.04"
  packages: [
    "bash", "python3", "tmux"
  ]
}

agents: {
  claude: { enable: true }
  codex: { enable: true }
}

export: {
  skills: ["agent-coach"]
  hooks: ["on-stop"]
}

build: {
  entry: "./run.sh"
  health_check: "./check.sh"
}

secret_keys: [
  "anthropic_api_key",
  "openai_api_key"
]
