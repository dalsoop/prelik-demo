01 · Declaration
.dalfactory 선언
레포 루트의 .dalfactory/templates/*.cue 에 달의 모든 구성 요소를 CUE 문법으로 선언합니다. CLI 도구, 스킬, 훅, 시크릿, 컨테이너 스펙 — 전부 한 파일.
dalcenter validate → CUE 스키마 즉시 검증
02 · Export
스킬·훅 내보내기
dalcenter export 실행 시 .dalfactory/skills/ 와 hooks/ 가 ~/.claude/skills/ 와 ~/.claude/hooks/ 로 심링크됩니다. claude의 settings.json 에 훅이 자동 병합.
codex 런타임은 ~/.codex/ 에 동일하게 적용
03 · Provision
컨테이너 자동화
Proxmox LXC 컨테이너가 자동 생성됩니다. CLI 도구 설치, PLAYER(claude/codex/gemini) 기동, SecretVault에서 AES-256-GCM 복호화 후 환경변수 자동 주입.
watch 60s → drift 감지 → reconcile 자동 실행
