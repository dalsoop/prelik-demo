⚙️ dalforge

하나의 dal이
모든 것을 담는다

dalforge의 핵심은 dal이라는 단위입니다. CLI, SKILL, HOOK, PLAYER, VAULT가 각각 고유한 DAL ID를 가지며 하나의 LXC 컨테이너에서 동작합니다.

  • ⚡
    DAL:CLI — dalcenter join 시 발급. 클라우드 허브 패키지 관리.
  • 🧩
    DAL:SKILL — .dalfactory/skills/ → ~/.claude/skills/ 심링크 export.
  • 🪝
    DAL:HOOK — 훅 스크립트 → settings.json 자동 머지.
  • 🤖
    DAL:PLAYER — claude_sdk / codex / gemini_cli LXC 내 실행.

🔐 VeilKey

시크릿은 절대
평문으로 남지 않는다

VeilKey는 AES-256-GCM으로 암호화된 SQLite에 자격증명을 저장합니다. dal 프로비전 시 자동으로 환경변수에 주입되어, 코드에 시크릿이 노출되지 않습니다.

  • 🔒
    AES-256-GCM 암호화 — 저장 즉시 암호화. 런타임에만 복호화.
  • 💉
    자동 env 주입 — provision 시 컨테이너에 자동 주입.
  • 🏰
    프로젝트별 격리 — 각 dal은 독립 Vault. 교차 접근 불가.
  • 👥
    팀 권한 관리 — 멤버별 시크릿 접근 권한 세밀 제어.
