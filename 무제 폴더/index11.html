What is a Dal?
달(dal)은 레포 하나에 귀속된 AI 에이전트 인스턴스. CLI 도구부터 스킬, 훅, PLAYER, 시크릿까지 하나의 단위로 선언·관리됩니다.
How it works
.dalfactory/templates/*.cue 에 선언 → dalcenter join · export · provision → 컨테이너 기동 + 런타임 심링크 자동화.
ID System
모든 구성 요소는 DAL:{CATEGORY}:{uuid8} 형식의 고유 ID를 가집니다. CLI · PLAYER · CONTAINER · SKILL · HOOK.
CLI
DAL:CLI:uuid8
달 안에 설치되는
명령줄 도구 패키지
CLI는 달 인스턴스 자체이자, 컨테이너 안에서 동작하는 도구 패키지입니다. cloud hub의 레지스트리에서 DAL:CLI ID로 버전 관리되며, provision 시 컨테이너에 자동으로 설치됩니다.

dalcenter join 명령이 레포를 달포지에 등록할 때 이 CLI 인스턴스 ID가 발급됩니다.
# .dalfactory/templates/default.cue cli: [ "DAL:CLI:b1c2d3e4", // dalcli-code-runner "DAL:CLI:f4a91c0e", // dalcli-test-gen ]
dalcenter join → DAL:CLI:a3f82c1d 발급
SKILL
DAL:SKILL:uuid8
AI 에이전트가 쓰는
스킬 파일 묶음
스킬은 .dalfactory/skills/ 안에 위치한 마크다운 기반 AI 지시 파일입니다. exports 에 선언하면 dalcenter export 실행 시 ~/.claude/skills/ 또는 ~/.codex/skills/ 에 심링크됩니다.

claude, codex 등 AI 런타임이 이 심링크를 통해 스킬을 자동으로 인식합니다.
# exports 선언 (templates/*.cue) exports: { claude: { skills: [ "skills/code-review/skill.md", "skills/pr-check/skill.md", ] } }
dalcenter export → ~/.claude/skills/ symlink
HOOK
DAL:HOOK:uuid8
git 이벤트에 연결되는
자동화 훅 스크립트
훅은 pre-commit, post-push 같은 git 이벤트에 연결되는 실행 스크립트입니다. dalcenter export 실행 시 ~/.claude/hooks/ 에 심링크되고, claude의 settings.json 에도 자동으로 등록됩니다.

달마다 독립된 훅 세트를 관리할 수 있어 레포별 AI 자동화가 가능합니다.
# exports.claude.hooks hooks: [ "hooks/pre-commit.sh", "hooks/post-push.sh", ] → ~/.claude/hooks/pre-commit.sh (symlink) → ~/.claude/settings.json (자동 병합)
dalcenter export → ~/.claude/hooks/ + settings.json
PLAYER
DAL:PLAYER:uuid8
컨테이너 안에서
실제로 실행되는 AI
PLAYER는 LXC 컨테이너 안에서 실행되는 실제 AI 에이전트입니다. claude_sdk, codex_appserver, gemini_cli, openai_compatible 타입 중 하나로 지정됩니다.

SecretVault에서 복호화된 시크릿을 환경변수로 받아 동작하며, watch 모드가 상태를 60초마다 점검합니다.
# container.agents (templates/*.cue) container: { base: "ubuntu-24.04" agents: { claude: { type: "claude_sdk" package: "@anthropic/claude-code" } } }
dalcenter provision → LXC 기동 + PLAYER 실행

01 · Declare
선언
.dalfactory/templates/*.cue 에 CLI · 스킬 · 훅 · 시크릿 · 컨테이너를 CUE로 선언. validate 즉시 검증.
$ dalcenter validate
02 · Register
등록 + 내보내기
join으로 DAL ID 발급. export로 스킬·훅을 ~/.claude/, ~/.codex/ 에 심링크. settings.json 자동 병합.
$ dalcenter join && export
03 · Provision
프로비저닝
LXC 컨테이너 자동 생성. CLI 도구 설치, PLAYER 기동, SecretVault 시크릿 복호화 주입.
$ dalcenter provision <id>
04 · Watch
자동 감시
60s 자동 점검. drift 감지 시 reconcile 자동 실행. 에이전트는 꺼지지 않습니다.
$ dalcenter watch
