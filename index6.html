Architecture

Hub-and-spoke hierarchy.

One KeyCenter controls policy. LocalVault nodes hold only ciphertext. Breach a node — the rest stay sealed.

CLI
Detection

222+ patterns.

AWS, GitHub, Stripe, private keys, JWTs, database URLs. Fully offline — no API needed.

Rotation

Zero-downtime rotation.

KeyCenter schedules. Each node re-encrypts independently. No restarts, no coordination.

Isolation

Every node, own key.

Each node has a unique DEK. Compromising one doesn't expose secrets on other nodes.

Encryption

Three-layer hierarchy.

1
KEK — from password, never stored scrypt
2
DEK — 256-bit, encrypted with KEK AES-256-GCM
3
Ciphertext — per-secret nonce VK:{REF}
Self-Hosted

Your infra. Your rules.

SQLite backend, no cloud dependencies. macOS (launchd) or Linux (systemd). Passwords from files, never env vars. AGPL-3.0 licensed — fork it, run it, own it.
