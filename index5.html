How It Works

Four steps to zero plaintext

1

Scan

Detect secrets in files with 222+ patterns. Fully offline.

vk scan .env
2

Filter

Replace plaintext with encrypted VK: references instantly.

vk filter .env
3

Wrap

Run commands with auto-masking. Secrets never leak to logs.

vk wrap ./deploy
4

Exec

Resolve VK: tokens at launch. Injected in memory, never disk.

vk exec ./app
Architecture

Every node isolated. Every key unique.

Hub-and-spoke hierarchy. Compromise a node — the rest stay sealed.
