Workflow

Four steps to sealed.

Every secret follows the same path. Detect, encrypt, mask, resolve.

1

Scan

222+ patterns. AWS, GitHub, Stripe, private keys, JWTs. Fully offline.

vk scan .env
2

Filter

Replace every detected secret with an AES-256-GCM encrypted VK: reference.

vk filter .env
3

Wrap

Execute commands with output masking. Secrets can't leak to logs or terminals.

vk wrap ./deploy.sh
4

Exec

Resolve VK: tokens at launch. Decrypted in process memory, never disk.

vk exec ./app
Features

Everything operators need.

🔐

Zero plaintext at rest

Encrypted at entry. Configs, git, logs — only VK: references survive.

↺

Seamless rotation

KeyCenter schedules, each node re-encrypts independently. No downtime.

⬡

Node isolation

Every node has its own DEK. Breach one — the rest stay sealed.

🤖

AI-safe references

AI agents see VK: tokens. They deploy, test, code — zero secret exposure.

🛡

Egress enforcement

Optional proxy blocks plaintext in outbound HTTP. Shell hooks block curl/wget.

📦

Self-hosted

SQLite, no cloud deps. macOS launchd or Linux systemd. AGPL-3.0.
